# cat-landcover-effects
Code accompanying the paper: "The effect of land cover on the home range of domestic cats"

# setting-up the environment:
set the python virtual environment
```bash
python3 -m venv .venv
source .venv/bin/activate
```

install the required packages
```bash
pip install -r requirements.txt
```

# running the code:
in order to fully reproduce the results of the paper, you will need to download the data from the following link: [data link]()


To run the code, execute the following command in the terminal:
```bash
python example.py
```
This will create an output directory with the geo-package layers for the human annotation step.
